# Trio Async Pipeline

## Overview
An educational Python project demonstrating async programming with the Trio library. The project implements a data-fetching and processing pipeline that covers 7 core Trio concepts: async/await, nurseries, concurrent execution, memory channels, error handling, coroutines, and structured concurrency.

## Project Structure
- `trio_pipeline.py` — Complete, runnable pipeline with extensive inline comments
- `TRIO_TUTORIAL.md` — Concept-by-concept tutorial walking through all Trio fundamentals
- `pyproject.toml` — Project metadata and dependencies

## How to Run
```bash
python trio_pipeline.py
```

## Key Dependencies
- Python 3.11+
- trio (async library)
